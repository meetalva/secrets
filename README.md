# secrets

> Shared secrets for Alva

## Prerequesites

* git
* [git-secret](https://git-secret.io/)
* [gpg](https://www.gnupg.org/)

## Getting started

```sh
git clone https://github.com/meetalva/secrets.git
cd secrets
git secret reveal # Assuming a core team member added your gpg key
```
